# Project Overview

This project, InterviewIQ, is designed to help users prepare for interviews.

# Setup

To get started with the project, follow these setup instructions:

1. Clone this repository:
   ```bash
   git clone https://github.com/anjaliOfficialcoll/InterviewIQ.git
   ```
2. Navigate to the project directory:
   ```bash
   cd InterviewIQ
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

# Usage

To run the project, use the following command:
```bash
npm start
```

# Features

- Feature 2: Feedback and improvement suggestions
